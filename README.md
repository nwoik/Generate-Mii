# Generate Mii

## Acquiring .rkg file
Visit [Ghost Manager](https://www.tt-rec.com/howto/?page=ghostmanager)

## Exporting Mii

## Viewing Mii
You can view your .miigx file on [mii-unsecure](https://mii-unsecure.ariankordi.net/)

Change the input type to `Mii data/file/Studio Code`
